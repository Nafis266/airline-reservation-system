# Airline Reservation System

This is a ReactJS + Supabase based Airline Reservation System project.

## Features
- User Signup and Login (Supabase Auth)
- View available flights and schedules
- Book flights with seat selection and class type
- Payment page (stores payment record in database)
- Admin Panel to manage flights, schedules, and crew

## Technologies Used
- ReactJS
- Supabase (PostgreSQL + Auth)
- JavaScript
- HTML / CSS

## Working
The frontend is built using ReactJS. Supabase is used as the backend database and authentication system. All flight booking, schedule management, crew assignment, and payment records are stored in Supabase tables.
